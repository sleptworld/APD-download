## 🧐 About <a name = "about"></a>

数据来源 https://www.quotsoft.net，脚本语言Python，只支持Python3.6以上版本。

## 🏁 Getting Started <a name = "getting_started"></a>

clone 仓库，在cmd或终端内输入 
```
python ./download.py
```

### Prerequisites

需要安装 rich、aiohttp 库

## 🎈 Usage <a name="usage"></a>

参数包括：

- 数据类型--type
- 时间范围-t （默认20150101-20201231）
- 下载路径-d （默认当前目录下的data文件夹）
